Approach: 0001 - Two Sums

Problem:

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

        Given nums = [2, 7, 11, 15], target = 9,
        Because nums[0] + nums[1] = 2 + 7 = 9,
        return [0, 1].

Approach: C++

Use a map to store the value and index of each element in the array. Then, for each element, check if the map contains the value that is needed to add up to the target. If it does, return the indices of the two elements.

Time complexity: O(n)

Space complexity: O(n)

Runtime: 8 ms, faster than 99.96% of C++ online submissions for Two Sum.

Memory Usage: 10.1 MB, less than 100.00% of C++ online submissions for Two Sum.

        
            class Solution {
            public:
                vector twoSum(vector& nums, int target) {
                    unordered_map map;
                    vector result;
                    for (int i = 0; i < nums.size(); i++) {
                        int complement = target - nums[i];
                        if (map.find(complement) != map.end()) {
                            result.push_back(map[complement]);
                            result.push_back(i);
                            return result;
                        }
                        map[nums[i]] = i;
                    }
                    return result;
                }
            };
